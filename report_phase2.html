Phase 2 태스크

구현 계획

4개 태스크 · 예상 총 작업 시간 2~3일

T1

RSS 뉴스 수집기

services/news_collector.py — feedparser 기반 멀티소스 수집 + 중복 제거

필수 신규 파일 ~0.5일

구현 항목

📌RSS 소스 목록 설정 (env/config)
📌feedparser로 병렬 피드 수집
📌URL 기반 중복 제거 (SQLite seen_urls 테이블)
📌최신 N시간 내 기사만 필터
📌NewsItem 스키마로 정규화 반환

코드 스케치

# services/news_collector.py async def collect_news(hours: int = 2) -> list[NewsItem]: feeds = [ "https://feeds.feedburner.com/CoinDesk", "https://cointelegraph.com/rss", "https://cryptonews.com/news/feed/", ] results = await asyncio.gather( *[_fetch_feed(url) for url in feeds] ) items = _deduplicate(_flatten(results)) return _filter_recent(items, hours=hours)
T2

Fear & Greed Index 서비스

services/fear_greed.py — alternative.me API 실시간 조회 + 캐싱

필수 신규 파일 ~0.5일

구현 항목

📌alternative.me API GET 요청
📌5분 캐싱 (불필요한 반복 호출 방지)
📌Mock 모드 지원 (랜덤값 반환)
📌/analyze 자동 주입 (요청에서 fear_greed_index 생략 시)
📌GET /api/v1/fear-greed 엔드포인트 추가

API 응답 예시

# GET https://api.alternative.me/fng/ { "data": [{ "value": "23", "value_classification": "Extreme Fear", "timestamp": "1748390400" }] } # 반환값 → scoring.py에 자동 주입 fear_greed = await get_fear_greed() # 23.0
T3

자동 분석 스케줄러

services/scheduler.py — APScheduler로 주기적 뉴스 수집 → /analyze 자동 실행

필수 신규 파일 ~1일

실행 흐름

⏰ APScheduler 트리거 (매 30분)
↓
📰 RSS 피드 병렬 수집
↓
😱 Fear&Greed Index 조회
↓
🧠 FinBERT 감성 분석
↓
📊 점수 계산 (뉴스 + 지표 + FG)
↓
💡 score ≥ 10: OpenClaw 알림 발송
↓
🤖 score ≥ 20: Bybit 자동 주문

스케줄 타임라인

매 30분
정기 분석

RSS 수집 → 감성 분석 → 점수 계산. 기본 주기. SCHEDULE_INTERVAL_MIN 으로 조정.

매 5분
Fear & Greed 캐시 갱신

외부 API 부하 최소화를 위해 캐시 TTL 5분으로 분리 운영.

매 1시간
seen_urls 정리

24시간 이상 된 중복 제거 기록 삭제. DB 용량 관리.

즉시
서버 시작 시 1회 실행

컨테이너 재시작 후 첫 번째 스케줄 전까지 공백 없이 즉시 분석.

T4

OpenClaw Webhook 수신 엔드포인트

api/routes.py 확장 — OpenClaw → trading-api 뉴스 Push 수신

권장 ~0.5일

신규 엔드포인트

POST /api/v1/webhook/news OpenClaw → trading-api 뉴스 push
GET /api/v1/fear-greed 현재 지수 + 캐시 TTL 반환
GET /api/v1/scheduler/status 스케줄러 상태 + 다음 실행 시각
POST /api/v1/scheduler/trigger 수동 즉시 실행 (테스트용)

Webhook 요청 형식

# OpenClaw → POST /api/v1/webhook/news { "token": "my_super_secret_token_123", "symbol": "BTCUSDT", "news": [ { "title": "Bitcoin ETF sees record inflows", "content": "...", "source": "coindesk" } ] } # 응답: 분석 결과 즉시 반환 { "score": 14.2, "action": "notify", ... }
